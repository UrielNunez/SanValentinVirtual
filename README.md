# SanValentinVirtual
Videojuego creado para San Valentín

Videojuego creado para San Valentín que presenta una carta en 3D como escena inicial, la cual se abre para revelar su contenido. Después de leer la carta, el jugador es invitado a elegir si quiere ser mi Valentín(a) con dos botones de 'Sí' y 'No'. Cada vez que se elige 'No', hay cambios en el texto y elementos de la escena, con la opción de cambiar de opinión. Una vez que se selecciona 'Sí', aparece una celebración en la pantalla. Desarrollado con Unity, utilizando Animator, C# Script, sprite editor y animator controller.
